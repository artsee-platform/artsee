## 1. 产品概述

**艺见心**是一款面向**艺术类留学申请者**的一站式服务平台，提供院校信息查询、作品集灵感参考、申请全流程管理、专家咨询及社区互动等核心功能。

平台同时提供 **Flutter 移动端 App**（iOS / Android）与 **Next.js Web 端**，数据共用 Supabase 后端。

---

## 2. 目标用户

| 用户类型 | 特征描述 |
|---------|---------|
| **核心用户** | 计划申请海外艺术类研究生（MFA、建筑、设计等）的本科毕业生/在读生 |
| **次要用户** | 国内艺术培训机构的规划老师、留学顾问 |
| **潜在用户** | 对艺术留学感兴趣的高中生、艺术爱好者 |

**核心用户画像（基于调研文档）：**
- 有本科艺术学位（如芝加哥大学视觉艺术 GPA 3.8/4.0）
- 目标：英国 G5 名校（牛津、剑桥、帝国理工+RCA、UCL、爱丁堡、中央圣马丁等）
- 需求：院校对比、作品集方向、文书写作指导、申请时间轴管理
- 年龄：22-28 岁，Z 世代，视觉审美偏好强

## 3. 产品定位与核心价值

**定位：** 艺术留学垂直赛道的「小红书 + 知乎 + 超级简历」

**一句话价值主张：** 让每一位艺术留学申请者都能找到方向、找到灵感、找到资源。

### 三大核心价值

```
全链路一站式服务              垂直艺术资源库               个性化申请规划
院校申请 · 作品集辅导           全球 Top 艺术院校库            定制选校方案
签证办理 · 导师咨询             名师作品集案例                背景提升路径规划
```

## 6. 核心功能模块

### 6.1 首页（发现流）

**功能描述：** 智能推荐信息流，聚合全站优质内容。

**核心功能点：**

- **混合推荐流**：根据用户关注的专业方向、目标院校，混合推送院校资讯、成功案例、热门帖子
- **Banner 轮播**：招生政策更新、截止日期提醒、平台活动
- **搜索入口**：全局搜索（院校名、专业、用户名）
- **通知中心**：消息图标，显示未读数

**交互规范：**
- 下拉刷新，上滑加载更多（无限滚动）
- 首次进入引导用户选择感兴趣的专业方向（冷启动标签选择）

---

### 6.2 探索（院校 & 课程）

**功能描述：** 全球艺术院校信息库，支持多维度筛选与深度解析。

#### 6.2.1 智能院校库

| 功能 | 详细说明 |
|------|---------|
| **多维筛选** | 按国家（英国/美国/德国等）、专业（纯艺/建筑/设计工程/插画等）、学费区间、院校排名、IELTS 要求精准筛选 |
| **院校卡片** | 展示院校 Logo、学校名（中英文）、专业名、学制、学费、IELTS 要求、申请截止日期 |
| **院校详情页** | 学校简介、课程设置、申请要求、推荐人数量、作品集要求、面试要求、校友案例入口 |
| **收藏对比** | 最多同时对比 4 所院校，生成对比表格 |

### 6.3 案例 & 灵感库

**功能描述：** 核心内容社区，双列瀑布流展示成功申请案例与作品集灵感。

#### 6.3.1 成功案例分享

| 功能 | 说明 |
|------|------|
| **瀑布流列表** | 双列瀑布流，每张卡片含封面图、标题、院校 tag、申请者头像 |
| **案例详情页** | 申请者背景（本科院校/GPA）、目标院校、录取结果、作品集图片/视频、申请心得文字 |
| **互动** | 点赞、收藏、评论、分享 |
| **标签体系** | 按专业（纯艺/设计/建筑）、院校、申请年份、国家筛选 |

**案例发布表单字段：**
```
标题
本科院校 & GPA
目标专业 & 院校
录取结果（录取 / 等候 / 拒绝）
作品集图片（最多20张）或视频（最长12分钟）
申请心得（富文本，可插图）
标签选择
是否匿名
```

#### 6.3.2 作品集灵感库

- 汇集高质量作品集图片与视频
- 支持按**专业、学校、风格、媒介（绘画/雕塑/摄影/装置/数字媒体）**标签精准筛选
- 全屏沉浸浏览模式（参考 Instagram）

#### 6.3.3 话题与挑战

- 运营发起热门话题（如「#我的伦敦求学故事」「#作品集改稿记录」）
- 创意挑战活动，激励用户参与内容创作

### 6.4 论坛（互动社区）

**功能描述：** 融合知乎问答沉淀 + 豆瓣小组圈层归属感的社区。

#### 6.4.1 问答模块（知乎模式）

| 功能 | 说明 |
|------|------|
| **提问** | 用户发起问题，可关联院校/专业标签 |
| **回答** | 支持富文本、图片、视频 |
| **赞同机制** | 高赞回答置顶 |
| **专业认证** | 优质答主认证为「导师」（需审核） |
| **关注问题** | 订阅问题更新通知 |

#### 6.4.2 话题小组（豆瓣小组模式）

- 按专业方向（纯艺组、建筑组、设计工程组）或目标国家（英国留学组、美国留学组）建立细分讨论区
- 用户可自由加入、发帖、置顶帖

#### 6.4.3 资讯推送

- 实时更新院校招生政策、截止日期变更、行业动态
- 用户可订阅关注院校的最新资讯

[2025.3-5]  递交申请

#### 6.5.4 导师咨询预约

| 功能 | 说明 |
|------|------|

## 10. UI 设计规范

参考《UI设计参考方案》，推荐以下设计方案：

### 10.1 色彩方案（推荐）

**方案 A：活力艺术橙（Vibrant Art Orange）**
- 主色：`#FF6A00`（橙色）
- 正文：`#333333`（深灰）
- 背景/卡片：`#FFFFFF`（纯白）
- 感觉：年轻、有活力、引人注目

**方案 B：沉静艺术青（Artistic Teal）**（备选）
- 主色：`#00A09D`
- 辅助：`#666666`
- 背景：`#F5F5F5`
- 感觉：专业、沉稳、值得信赖

### 10.2 字体系统（推荐）

**方案 A：现代几何无衬线**
- 英文：`Poppins`（标题 Bold / 正文 Regular / 说明 Light）
- 中文：`思源黑体（Source Han Sans）`
- 感觉：现代感 · 国际范 · 清晰易读

### 10.3 布局规范

- **卡片式 + 充足留白**：降低视觉负担，模块化展示
- **网格系统**：严格对齐，8pt 基础栅格
- **双列瀑布流**：案例/灵感库页面

## 12. 非功能性需求

### 12.1 性能

- 首屏加载时间 ≤ 2s（4G网络）
- 图片懒加载 + CDN 加速（Supabase Storage）
- 案例列表分页：每次加载 20 条

### 12.2 安全

- Supabase RLS（行级权限）：用户只能修改自己的数据
- 内容审核：发布内容经过关键词过滤 + 人工抽查
- 文件上传：限制格式（JPG/PNG/MP4/PDF）与大小（图片≤10MB，视频≤500MB）

### 12.3 隐私

- 案例支持匿名发布
- 个人 GPA、院校背景信息用户自主选择是否公开

### 12.4 可访问性

- 图片必须提供 Alt 文本
- 字体最小 12pt，主要正文 14-16pt
